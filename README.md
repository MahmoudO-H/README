# README
Titre: Minecraft stéréotypes
Description: Ce projet est fait dans le but de présenter d'une manière humouristique des stéréotypes du jeu Minecraft. Notre concepte est de faire l'utilisateur utiliser une télé commande pou naviguer les différents stéréotpes. L'intention artistique est de faire le site ressembler à Minecraft.
